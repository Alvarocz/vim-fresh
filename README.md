# Installation
```sh
$ git clone https://github.com/Alvarocz/vim-fresh.git && cd vim-fresh
```

## Using vim
```sh
$ cp colors/fresh.vim ~/.colors/ (or whatever directory you use)
```

## Using neovim
```sh
$ cp colors/fresh.vim ~/.config/nvim/colors/
```
